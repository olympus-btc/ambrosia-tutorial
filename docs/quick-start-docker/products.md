# Managing Products

## 1. Open the Products Page
- From the Dashboard → **Products**
- You’ll see the list of existing products (name, price, stock, actions).

## 2. Add a Product

### Step 1 — Click "Add Product"
Top-right button.

### Step 2 — Fill the fields

**Product Name**
```
5 Sticker Pack
```

**Description**
```
Pack with 5 random stickers
```

**Category**  
Create a new category.

```
Stickers
```

**SKU**
```
PK-STICK-05
```

**Price (USD)**
```
1
```

**Stock**
```
20
```

**Image (optional)**  
Upload a PNG/JPG image (square recommended).

### Step 3 — Click “Add”
Product appears immediately in the list.
