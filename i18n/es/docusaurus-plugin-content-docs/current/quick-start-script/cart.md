# Procesamiento de Ventas

## 1. Abre el Carrito
- Dashboard → **Carrito**
- Izquierda: Productos
- Derecha: Resumen del Pedido + Pago

## 2. Agrega Productos

Haz clic en **Agregar** en cualquier tarjeta de producto.

## 3. Ajusta o Elimina Artículos

**Aumentar cantidad:** ▲

**Disminuir cantidad:** ▼

**Eliminar:** Ícono de papelera roja

Los totales se actualizan automáticamente.

## 4. Revisa el Pedido
Verifica:
- Artículos
- Cantidades
- Monto total

## 5. Selecciona el Método de Pago
Elige del menú desplegable:
- **BTC (Lightning)**

## 6. Completa el Pago

### Bitcoin (Lightning)
1. Selecciona **BTC**
2. Haz clic en **Pagar**
3. El cliente escanea el código QR de la factura
4. Una vez pagado, haz clic en **Pago Recibido**
5. Éxito → El carrito se vacía
