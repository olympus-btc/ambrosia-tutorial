# Gestión de Productos

## 1. Abre la Página de Productos
- Desde el Dashboard → **Productos**
- Verás la lista de productos existentes (nombre, precio, stock, acciones).

## 2. Agregar un Producto

### Paso 1 — Haz clic en "Agregar Producto"
Botón en la parte superior derecha.

### Paso 2 — Completa los campos

**Nombre del Producto**
```
5 Sticker Pack
```

**Descripción**
```
Pack with 5 random stickers
```

**Categoría**
Crea una nueva categoría.

```
Stickers
```

**SKU**
```
PK-STICK-05
```

**Precio (USD)**
```
1
```

**Stock**
```
20
```

**Imagen (opcional)**
Sube una imagen PNG/JPG (se recomienda formato cuadrado).

### Paso 3 — Haz clic en "Agregar"
El producto aparece inmediatamente en la lista.
