# Tutorial de Ambrosia

Un tutorial para instalar Ambrosia (Punto de Venta), configurarlo y usarlo.

## ¿Qué es Ambrosia?

**Ambrosia** es un moderno sistema de Punto de Venta (PoS). Permite a los negocios aceptar pagos en Bitcoin a través de Lightning Network mientras gestionan inventario, ventas y operaciones del negocio en un sistema unificado.

## Lo que Aprenderás

Siguiendo este tutorial, podrás:
- Instalar y configurar Ambrosia en tu máquina
- Gestionar usuarios y productos
- Procesar tu primer pago Lightning
- Abrir y cerrar un canal Lightning

## Requisitos

### Costo:
- Si solo usas una **laptop**
  - ~$GRATIS

### Hardware

#### Laptop
- Funciona en Linux, Mac o (¡incluso!) Windows
- Úsala para aprender, aunque ten en cuenta que necesitarás algo que puedas dejar encendido y conectado a internet 24/7 si realmente quieres usarlo en una tienda.

### Requisitos de Software

Antes de comenzar, debes tener familiaridad básica con:
- Usar una terminal/línea de comandos
- Navegación básica de archivos
- Instalar software en tu sistema operativo

¡No te preocupes si eres nuevo en Lightning Network o Bitcoin — te guiaremos en todo!
